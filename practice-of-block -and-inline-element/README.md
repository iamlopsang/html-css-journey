## Simple Navigation Bar
    A clean and simple navigation bar built using basic HTML and Css. This project focuses on understanding inline, block, and inline-block elements along with CSS pseudo-classes.

## Features
    > Horizontal navigation menu
    > Styled links using inline-block
    > Hover, focus, and active effects
    > Cleaning space using margin and padding

## Purpose of the project

    > Inline vs block vs inline-block behavior
    > Text alignment using (text-align)
    > Spacing using margin and padding
    > Styling links with pseudo-classes (:hover, :active, :focus)

## Built with
    > HTML
    >Css
