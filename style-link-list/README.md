## Styled Link List
    A simple project that demonstrates how to style lists and links using HTMLand CSS. This project focuses on understanding list behavior and link pseudo-classes.


## Features
    > Styled unordered list
    > Custom spacing between list items
    > Links styled as buttons
    > Hover, focus, active, and visited effects
    > Clean and centered layout

## Purpose of the project
    This project was created to practice core CSS concepts:
        > Styling lists using <ul> and <li>
        > Applying spacing with margin and padding
        > Using link pseudo-classes(:link, :visited, :hover, :focus, :active)

## Acknowledgement
