Learning about Lists in Html

Types of Lists

Unordered List

An unordered list is a collection of items that do not have a specific order. It is created using the <ul> tag.

  • Item one
  • Item two
  • Item three

Ordered List

An ordered list is a collection of items that are arranged in a specific order. It is created using the <ol> tag.

items needed to bake a cake

  1. Egg
  2. Milk
  3. Flour
  4. Water
  5. Coco Powder

Definition List

A definition list is a collection of terms and their definitions. It is created using the <dl> tag.
